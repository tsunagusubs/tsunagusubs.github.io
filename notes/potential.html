potential form (b4 + る / b2 + られる)

japanese iii / 14 february 2017


- conjugation -

Class I: B4 (Base 4) + る


Remember, to change a Class I verb's base to 4, you must first look at the last hiragana character. Now replace it with a character on the same line that makes an え sound. For example, とぶ (jump/fly) becomes とべ and 読む (read) becomes 読め.

To make it potential (able to do) form, simply add る to a Class I base 4 verb.

Class II: B2 (Base 2) + られる

Drop the る from the Class II verb and add られる.
